# US-1 : Rédaction assistée par IA

## Description
En tant qu'utilisateur, je veux générer automatiquement du texte à partir d'un prompt,
afin de gagner du temps dans la rédaction de documents.

## Critères d'acceptation
- [ ] L'utilisateur peut saisir une consigne (ex : "écris un mail professionnel")
- [ ] Le système génère un texte cohérent en moins de 5 secondes
- [ ] L'utilisateur peut copier ou insérer le texte généré
