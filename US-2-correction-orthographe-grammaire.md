# US-2 : Correction orthographe et grammaire

## Description
En tant qu'utilisateur, je veux corriger automatiquement mes textes,
afin d'améliorer leur qualité.

## Critères d'acceptation
- [ ] Le système détecte les fautes d'orthographe et de grammaire
- [ ] Les corrections sont proposées clairement
- [ ] L'utilisateur peut accepter/refuser chaque correction
