# US-3 : Résumé de documents

## Description
En tant qu'utilisateur, je veux obtenir un résumé d'un document,
afin de comprendre rapidement son contenu.

## Critères d'acceptation
- [ ] L'utilisateur peut importer ou coller un texte
- [ ] Le résumé est plus court que le texte original
- [ ] Les idées principales sont conservées
