# US-7 : Traduction multilingue

## Description
En tant qu'utilisateur, je veux traduire un texte dans une autre langue,
afin de communiquer avec des interlocuteurs internationaux.

## Critères d'acceptation
- [ ] L'utilisateur choisit la langue cible
- [ ] La traduction est fidèle au texte d'origine
- [ ] Le système supporte au moins 3 langues
